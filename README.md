# Cypress Tests

This project contains Cypress end-to-end tests.

## Running Cypress

- **Run Cypress with UI (debug mode):**

  ```bash
  npx cypress open
  ```

- **Run Cypress in headless mode:**

  ```bash
  npx cypress run
  ```
