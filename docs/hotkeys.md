# VS Code Hotkeys

## Search

- **Project search:** `Ctrl + Shift + F`
- **File search:** `Ctrl + P`

## Editing

- **Duplicate selection:** `Shift + Alt + Down` (Windows)
- **Reformat code:** `Shift + Alt + F`

## Navigation

- **Go to Definition:** `F12`
- **Navigate Back:** `Alt + Left Arrow`
- **Navigate Forward:** `Alt + Right Arrow`
